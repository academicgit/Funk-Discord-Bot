<h2>🛠️ Installation Steps:</h2>

<p>1. Cloning</p>

```
git clone https://github.com/academicgit/Kalu_Bhaiya_Discord_Bot.git
```

<p>2. Install Packages</p>

```
pip3 install -r requirements.txt
```

<p>3. Declare Token as ENV</p>

```
export TKN=yourtokenhere
```

<p>4. Run</p>

```
python main.py
```
